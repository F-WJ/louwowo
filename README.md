# 骡窝窝游记项目
---
### 功能块
1. 登录和注册
2. 后台攻略，游记
3. 微信评论式（mongodb）
4. 数据统计（阅读数，点赞数。。。）（Redis）
5. 攻略排行榜（Redis）


### 后端
